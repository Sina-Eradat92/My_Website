CSUN Accessibility compotation:

As a random five person team we developed a text to Unicode converter app to help those with visual disabilities. The app would allow the user to enlarge the text and through lexicographic analysis read the text to the user. The compotation taught me how to work under strict time constraints, as well as scrum project management.

  • Contribution: Image ingestion and preprocessing, and text read back functionality.
  • Utilized: C and Android programming.
