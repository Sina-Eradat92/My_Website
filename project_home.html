Projects

CSUN Accessibility Competition:

A competition hosted by the Computer Science department at California State University, Northridge to spark innovation. As a five person team, we developed a mobile application whose purpose was to help those with auditory, visual and mobile disabilities. The app used lexicographic analysis to convert pictures into text that the user could enlarge or have read to them. The competition taught me the importance of developing work under strict time constraints as well as implementing scrum project management throughout the course of this project.

  • Personal contribution: Image ingestion and preprocessing, and text read back functionality.
  • Utilized: C and Android programming.

Umbrella Irrigation:

Updated the California State University, Northridge irrigation system by integrating a new Raspberry Pi System with an intuitive custom Web UI. Going into this project I had very little knowledge about microcontroller or firmware development, but as the project progressed I quickly learned the fundamentals of ARM based development as well as SPI and Socket communication protocols.

  • Personal Contribution: SPI protocol for expanded I/O, Socket/ (IP/ TCP) programming for communication to a web based server
  • Utilized: C programming, STM32 microcontroller, Raspberry Pi 3, Circuit Design
Machine Learning

I have always been interested in automation and how humans can train computers to perform tasks with extreme precision. From simple AI that takes the role of enemies in a video games to self driving cars and fully autonomous robots, I am passionate about this field because it gives me a reason to continue learning and testing out new ideas. I have worked on the following data sets: MNIST, Titanic, and CIFAR. My future plans include: training models using openCV and live video.

  • I have started participating on Kaggle in order to further my understanding of machine learning
  • Utilized: Python, Tensorflow, Keras, Scikit-learn, Pandas, openCV
